LIBPERIF
========

LIBPERIF is an AVR C library for controlling peripheral devices.

###Supported Devices

* 74HC595 (serial-to-parallel shift register)
* 74LS165 (parallel-to-serial shift register)
* I2C, SPI, USART devices
